ubuntu下安装sublime并运行php

首先下载源码到指定目录

#cd /usr/local/src
#wget http://c758482.r82.cf2.rackcdn.com/sublime_text_3_build_3083_x64.tar.bz2
#tar -jxvf sublime_text_3_build_3083_x64.tar.bz2
#cd sublime_text_3/
#./sublime_text

就这样，安装好了，但是如果我们向讲sublime固定在启动器里时，退出后，又消失了

#vim sublime_text.desktop //把目录设置成实际的目录，并将Icon设置为实际图片地址，我这里是/usr/local/tools/sublime_text_3/Icon/48×48/sublime-text.png
#cp ./sublime_text.desktop /usr/share/applications/

这样，在终端执行./sublime-text后，启动器里就有sublime的图标，右键锁定即可。
接下来，是运行php，打开sublime text后，tools->build system->new build system，在新打开的文件里，输入

{ "cmd": ["/usr/local/bin/php" ,"$file"], "file_regex" : "php$", "selector" : "source.php", }

然后点击保存，但是，可能在保存的框里，系统并没有选中任何文件夹。这个时候，打开perference ->browse packages，应该会报权限不足，进入终端

$chmod -R 0777 ~/.config/sublime-text-3

重新保存一下，会默认选中～/.config/sublime-text-3/Packages/User，ok，运行php搞定了。
