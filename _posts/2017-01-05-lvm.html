使用lvm管理逻辑分区

通过使用lvm来管理逻辑分区，linux下就可以很方便的扩容。示意图：


这张图简单的描述了硬盘、分区、卷组、lv之间的关系，理解清楚后，对之后的操作就不会有疑问了。接下来是具体的操作：

pengzhen@pz:~$ sudo apt-get install lvm2 //安装lvm管理软件
pengzhen@pz:~$ sudo fdisk /dev/sda

我在虚拟机中操作，遇到了报错信息 Re-reading the partition table failed.: Device or resource busy。重启虚拟机就可以了，如果没报错，则执行 partprobe命令，让内核重新读取分区信息.
接下来就是创建卷组，创建lv，挂载就行了。

root@pz:~# pvcreate /dev/sda4 //转成物理卷(pv)
root@pz:~#:~$ vgcreate VolGroup_0 /dev/sda4 //加入卷组
root@pz:~# vgdisplay
